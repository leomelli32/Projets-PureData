## Avant-propos

Si vous voulez manipuler mes programmes, veuillez suivre certaines instructions au préalable :
- Installer la libraire GEM dans le registre "Outils" - "Installer des objets supplémentaires": je l'utilise beaucoup, elle permet d'ouvrir une nouvelle fenêtre et de voir des modélisations
- Activer le DSP dans le menu en haut, dans le registre "Média"; VEUILLEZ METTRE VOTRE SON AU MINIMUM, les sons peuvent parfois être très puissants
- Si vous êtes au casque, veuillez configurer le périphérique de sortie dans "Média" - "paramètres audio"
